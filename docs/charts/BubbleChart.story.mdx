## Quick Start
To create a bubble chart, use import the `BubbleChart` and give it `data`. The chart
will automatically configure itself with the default options exposed via `props`.

```jsx
import { BubbleChart } from 'reaviz';

const MyChart = () => (
  <BubbleChart
    height={300}
    width={300}
    data={[
      { key: 'AWS', data: 13 },
      { key: 'SendGrid', data: 2 },
      { key: 'Okta', data: 15 }
    ]}
  />
);
```

## Demo

```js codesandbox=reaviz
import React from 'react';
import { BubbleChart } from 'reaviz';

const data = [
  { key: 'AWS', data: 100 },
  { key: 'SendGrid', data: 45 },
  { key: 'Okta', data: 75 },
  { key: 'Twillo', data: 25 }
];

export default () => (
  <div style={{ margin: '55px', textAlign: 'center' }}>
    <BubbleChart
      data={data}
      height={450}
      width={450}
    />
  </div>
);
```
